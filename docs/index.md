# Whisp Developer Documentation

Whisp is a scalable chat infrastructure you can integrate into web, iOS, and Android apps using SDKs and APIs.

::scalar-button{title="Start with the JS/TS SDK" href="/sdks/js" icon="phosphor/regular/package"}
::scalar-button{title="Open REST API Reference" href="/api" icon="phosphor/regular/brackets-angle"}

:::scalar-callout{type="warning"}
Never expose your Whisp API key in browser/mobile code. Use your backend to authenticate and pass short‑lived tokens to clients.
:::

## What you’ll find here

- **SDK guides:** installation, quickstarts, auth, realtime, errors
- **REST API reference:** generated from your OpenAPI file
- A structure that scales as you add more SDKs
